# 📰 Monitor de Notícias de Calcário

Um sistema automatizado de monitoramento de notícias sobre calcário agrícola, cal agrícola e corretivos de solo.

## 🎯 Funcionalidades

- ✅ Coleta automática de notícias de múltiplas fontes
- ✅ Filtra notícias por palavras-chave
- ✅ Gera relatório em Excel
- ✅ Publica página web moderna e responsiva
- ✅ Atualiza automaticamente via Netlify
- ✅ Design moderno com gradiente e efeitos hover

## 🔑 Palavras-chave

- Calcário agrícola
- Cal agrícola
- Calcário
- Corretivo de solo
- Limestone
- Agricultural lime

## 📦 Requisitos

- Python 3.8+
- Bibliotecas: requests, pandas, beautifulsoup4, openpyxl, lxml

## 📊 Saídas

- `index.html` - Página web com as notícias
- `Noticias_Calcario_YYYYMMDD.xlsx` - Relatório em Excel

## 📅 Atualizações

As notícias são coletadas dos últimos 360 dias.
